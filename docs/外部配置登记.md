# 外部配置登记

本文档记录当前工作台依赖的外部地址、本机路径和环境变量。登记只用于维护识别，不代表可以直接修改默认值；任何配置化都必须逐条处理，并保留现有默认值回退。

## 配置化原则

- 先登记，后改代码；不得一次性替换所有默认值。
- 禅道、任务同步、拖拽指派、附件代理和摘要生成属于高影响链路，修改前必须单独回归。
- Google Sheet、组员上报地址、Git 资源仓库和本机工具路径可以逐步环境变量化，但必须保留现有默认值。
- 密码、Token、API Key 不写入本文档真实值；只登记变量名和用途。
