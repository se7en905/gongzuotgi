# 美术简报识别规则

来源：`/Users/se7en/000group/禅道任务2.0/禅道美术简报`  
原产物作者：`lihualing`

## 识别来源

- 任务标题。
- 任务描述。
- 关联需求说明。
- 验收标准。
- 评论补充。
- 正文图片和附件图片。

## 美术相关关键词

- 美术、效果图、UI、ui、图标、金币、锁住、锁定、未解锁。
- 蒙版、提示条、tips、tip、弹层、按钮、卡片。
- 展示、样式、切图、资源、视觉、客户端、web。
- 尺寸、格式、大小限制、间距、字号、Banner、图片、短文案。

## 开发和后台关键词

- 接口、后台、校验、发放、派发、字段、保存、回显、自动、逻辑、数据库、统计口径。
- 配置、启停、启用、关闭、开启、排序、跳转、内链、外链、上传、维护、选择、开关、运营。
- 鼠标移入、鼠标移出、移入时、未移入、预览按钮、点击弹出预览。

## 不做项识别

包含以下语义的原文应进入“不需要美术做”：

- 不要。
- 不需要。
- 不调整。
- 不做。
- 无需。
- 不再。

## 特殊推断

- 出现“金币”时，应提醒确认是否沿用现有金币图标、单位和领取反馈。
- 出现“锁住、锁定、未解锁、门槛、蒙版”时，应提醒确认锁住态覆盖范围。
- 出现“新增入口、默认图标、入口名称”表格时，应提取入口名称并提醒补齐默认图标。
- 出现“黑色 tips / 气泡弹层不做”等表述时，应明确写入“不需要美术做”。

## 图片筛选

- 客户端、游戏、Web 前台视觉截图进入参考图主区。
- 后台表格、配置页、低色彩管理后台截图进入过滤区。
- 下载失败的图片保留 URL 和错误原因。

## 输出原则

- 不凭空扩展需求。
- 不把开发配置误判为美术产出。
- 不删除“不做”原意。
- 对边界不清的内容标为“需要确认”。
