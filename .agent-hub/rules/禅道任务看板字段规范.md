# 禅道任务看板字段规范

来源：`/Users/se7en/000group/禅道任务2.0/禅道任务提醒`  
原产物作者：`lihualing`

## 任务基础字段

| 字段 | 含义 | 工作台用途 |
|---|---|---|
| `id` | 禅道任务 ID | 卡片主编号、打开任务、去重 |
| `name` | 任务标题 | 卡片标题、复制标题 |
| `status` | 任务状态 | 状态标签、筛选 |
| `pri` | 优先级 | 排序和风险判断 |
| `type` | 任务类型 | 辅助判断工作类型 |
| `estimate` | 预计工时 | 工时风险参考 |
| `consumed` | 已耗工时 | 工时进度 |
| `left` | 剩余工时 | 剩余投入判断 |
| `deadline` | 截止日 | 逾期、今天、明天、即将到期分组 |
| `estStarted` | 预计开始 | 排期参考 |
| `assignedDate` | 指派时间 | 新指派和响应时效判断 |
| `storyID` | 关联需求 ID | 追溯需求 |
| `storyTitle` | 关联需求标题 | 简报上下文 |

## 执行字段

| 字段 | 含义 | 工作台用途 |
|---|---|---|
| `executionId` | 执行 ID | 标识所属执行 |
| `executionName` | 执行名称 | 卡片副标题 |
| `executionStatus` | 执行状态 | 扫描范围过滤 |
| `executionBegin` | 执行开始日期 | 项目周期参考 |
| `executionEnd` | 执行结束日期 | 项目周期参考 |

## 默认筛选

- 执行状态：`doing`、`wait`。
- 任务状态：`wait`、`doing`、`pause`。
- 排除已删除任务。
- 支持递归子任务。

## 默认排序

1. 截止日升序。
2. 同一天按任务 ID 升序。
3. 看板风险展示时，逾期优先于今天，今天优先于明天。

## 卡片动作

- 打开：禅道任务详情页。
- 工时：任务登记工时页。
- 备注：任务编辑或备注页。
- 复制标题：复制 `name`。
- 美术：根据 `id` 调用美术简报技能。
