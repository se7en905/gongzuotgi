文字版速读

先分 4 类看

平台里的“删除”不是一回事，主要分成：硬删除业务记录、删除执行档案和目录、软删除/隐藏、维护中心范围清理。

最危险的不是普通删除

真正要谨慎的是会动 workspace、artifactRoot、生图目录、执行归档目录的入口，这类会删历史材料和结果证据。

项目删除不是全删

删除项目只会清项目接入、同步任务/Bug、扫描缓存等平台数据，不会顺手删除执行记录、自定义流程模板和累计调用次数。

很多“删除”其实是隐藏

人工研究记录、验证回填、项目字段/项目行这几类，本质是给工作台加隐藏标记或删除标记，不会回写 Google 表或外部源数据。

操作日志删除不回退统计

操作日志范围删除只删日志本身，不会影响任务、模板、执行记录，也不会扣减 AI 产物累计调用次数。

AI 全流程人工记录是软删除

这类删除当前是把记录状态改成 deleted，前端不再展示，并不是把底层数据文件里的记录物理移除。

低风险：隐藏 / 作废

主要影响工作台展示或评分口径，不改外部源数据，不删原始文件。

  • 人工研究记录隐藏
  • 验证回填删除
  • 项目字段/项目行隐藏
  • 产物作废

中风险：业务记录硬删除

会真删平台内业务记录，但通常不碰执行档案目录和累计指标。

  • 平台任务删除
  • 项目删除
  • 模板删除
  • 账号/角色删除

高风险：执行档案删除

不止删一条记录，还会清掉工作区、执行产物或归档目录。

  • 单条执行记录删除
  • AI档案范围删除
  • 生图产物范围清理

最高注意：维护中心清理

按范围批量生效，适合清理但也最容易误删整批历史材料。

  • 安全维护清理
  • 操作日志范围删除
  • 摘要范围清理
  • 执行历史范围清理

负责人判断口诀

先看是不是“隐藏”

只要文案里明确写了“不会删除源文件 / 不会修改 Google 表 / 只影响工作台展示”，这类一般先归到低风险的展示层隐藏。

再看会不会动目录

只要涉及 workspace、artifactRoot、生成图片目录、执行归档目录，这类就不是普通删记录，而是执行档案删除。

批量范围删除优先谨慎

维护中心、AI档案范围删除、生图范围清理这几类都应当按“先预览，再确认，再复核保留项”的管理方式处理。

界面按钮逐项说明

这一节按“某个界面里的某个删除按钮”来写，重点回答 5 件事：只是前端不显示，还是后端数据销毁；会不会删目录/文件；会牵连哪些页面；明确不影响哪些页面。

界面 / 按钮 删除层级 实际删除范围 会牵连的面 明确不影响
AI档案
删除当前范围		 后端记录销毁 目录一起删 删除当前筛选命中的执行记录；同时删除这些记录对应的执行工作区和产物归档目录。不是单纯前端隐藏。 AI档案列表会少这些记录；美术执行台里同一批 run 也会消失；对应过程材料、执行附件、执行产物以后无法再预览或下载。 不会删除任务中心任务；不会删除禅道任务；不会回退 AI产物累计调用次数；不会删项目接入记录和模板。
执行台
删除执行记录		 后端记录销毁 目录一起删 删除当前这条 run 记录，同时删除它的工作区和 `artifactRoot` 产物目录。不是只清前端内存。 执行台当前记录消失；AI档案里对应明细也消失；该执行的日志、过程材料、产物证据不再可追。 不会删除任务中心任务；不会删除禅道任务；不会删除其它执行记录；不会影响累计调用统计。
任务中心
删除平台创建任务		 后端记录销毁 删除平台任务本身，以及和这条任务关联的任务处理备注、验收记录、美术摘要索引。不是只把列表这行隐藏。 任务中心该单据消失；关联备注、验收、摘要入口一并清掉；如果当前正查看该任务详情，会失去对应面板内容。 不会删除禅道同步任务；不会删除禅道原任务；不会删除已产生的执行档案和执行产物。
项目接入 / 扫描来源
删除项目		 后端记录销毁 删除项目接入记录、平台内同步任务/Bug、相关备注/验收/摘要索引、扫描缓存。不是删本地项目文件夹。 项目列表消失；任务中心里该项目下的平台同步任务和 Bug 消失；AI产物扫描来源和项目扫描缓存一起失效。 不会删除本地项目目录；不会删除执行记录；不会删除自定义流程模板；不会删除累计调用次数。
操作日志
删除当前范围		 后端记录销毁 只删除命中的操作日志记录；没有目录清理；不是仅前端不显示。 操作日志页少掉这些审计记录；研究同步里的“接入测试/操作日志联动展示”如果依赖该日志，后续也查不到原日志源。 不会删除任务、执行记录、模板、项目；不会扣减累计调用次数；不会删除产物文件。
研究同步
删除单条 / 删除某 Skill 下全部		 后端记录销毁 删除 `artProgressEvents` 研究同步记录本身；不删外部文件。 研究同步列表、相关汇总、按 Skill 聚合视图都会少这批记录；若验证回填或统计依赖这些研究展示，会失去这部分展示来源。 不会删禅道；不会删组员本机文件；不会删执行记录；不会删 Google 表源数据。
人工研究清单
隐藏 / 删除人工研究记录		 仅工作台隐藏 本地写一条 override 删除标记，让这条记录不再展示；不是物理删除源数据。 人工研究清单里该记录隐藏；如果当前统计口径依赖“可见记录”，页面展示会跟着变化。 不会修改 Google 表源数据；不会删除文件；不会删共享盘内容；不会删执行档案。
验证回填
删除		 仅工作台隐藏 给验证回填记录写删除标记，工作台不再展示；不是回写删 Google 表原始行。 验证回填列表会少这条；相关验证展示、匹配结果、基于当前记录的工作台视图会同步少一条。 不会修改 Google 表源数据；不会删本地产物文件；不会删任务和执行记录。
项目字段表
隐藏字段		 仅前端展示隐藏 字段不再展示，已有行数据保留；不做业务数据销毁。 项目字段表界面少一个字段；依赖该字段显示的局部视图不再显示该列。 不会删行数据；不会改外部源表；不会删其它业务记录。
项目字段表
隐藏项目行		 仅工作台隐藏 记录隐藏标记，避免外部刷新后又出现；不是真删源行。 项目字段表里这行消失；相关本地展示不再出现该行。 不会删除外部表原始行；不会删除其它页面业务数据。
AI产物清单
作废产物		 口径作废 把这条产物从有效统计 / 评分口径里作废，不是删除源文件，也不是删除记录目录。 AI评分、有效占比、清单展示口径会变化；相关统计卡会少掉这条有效产物贡献。 不会删除源文件；不会删共享盘文件；不会删执行档案；不会删调用日志。
AI全流程人工记录
删除		 软删除 把记录状态改成 `deleted`，前端过滤掉；底层记录还在文件里，不属于物理销毁。 AI全流程人工记录列表不再显示；相关人工补录视图会少一条。 不会删除 run；不会删任务；不会删目录和产物归档。
维护中心
安全维护清理		 文件清理 非业务删单 只删 `.DS_Store`、过期 JSON 临时文件、已无 run 对应的孤儿工作区。不是删业务记录。 磁盘垃圾减少；维护中心概览体量变化。 不会删业务 JSON；不会删有效执行工作区；不会删正式产物；不会删任务、模板、AI档案。
维护中心
生图产物清理		 产物引用变更 图片目录删除 删除执行归档里的“生成图片”目录，并从 run 记录里删掉对应图片引用；run 本身保留。 执行台和 AI档案里这些图片以后无法预览、打开、下载；同一执行记录仍在，但成品图区会空掉或减少。 不会删除执行记录本身；不会删任务；不会删禅道；不会回退累计调用指标。
维护中心
摘要范围清理		 索引销毁 摘要目录删除 删除 `outputs/art-briefs/` 命中的摘要目录，并同步删摘要索引记录。不是只清界面缓存。 任务中心/摘要入口以后看不到这些摘要 HTML 和 AI 工作说明；相关摘要列表记录一并消失。 不会删除禅道任务；不会删除任务中心任务本身；不会删除执行记录。
